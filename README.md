## Installation

#### Vue Configuration

```sh
    cd eruca
```

```sh
    npm install
```


```sh
    npm run watch
```
#### Laravel Configuration


```sh
    composer install
```

```sh
    php artisan key:generate
```

```sh
    php artisan migrate
```

```sh
    php artisan serve
```
